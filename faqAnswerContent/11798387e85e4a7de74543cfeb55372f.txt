To check out a license, please use the following steps:


1. Start > All Programs > CATIA (or CATIA PLM Express)> Tools > Nodelock Key Management (DSLS) V5-6R20XX
2. Click Offline Management Tab

3. Click the RefreshList button

4. Select the licence off the list you wish to check out and Right-Click > Extract

5. Set the number of days and click OK

6. Click OK
