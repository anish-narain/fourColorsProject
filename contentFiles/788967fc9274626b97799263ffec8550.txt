This short guide lists the steps to enrolling a LUM nodelock license.




Please copy the licences received onto the computer
1. Go to C:ProgramDataIBMLUM
2. Rename NODELOCK to NODELOCK.bak (if NODELOCK file is missing then ignore this step).

3. Go to Start > All Programs > CATIA > Tools > Nodelock Key Management (LUM) V5XXXX
4. Click OK

5. Click File > Import

6. Select File and click Open
